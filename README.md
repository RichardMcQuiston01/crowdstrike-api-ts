# TYPESCRIPT HELPER FOR CLOUDSTRIKE API

## DESCRIPTION

A package that can be integrated into other TypeScript applications for interacting with the CloudStrike API.  For example, an Electron desktop app that needs to interact with the API to get data for displaying to the User.  Or, a backend API service that needs to make API calls to retrieve and analyze data from their CloudStrike instance.

## Guidelines

0. **Always use `bun`, not `npm`.**
1. Code must be well-written and well-documented using modern coding standards.
2. Code must follow the Google TypeScript Style guide unless otherwise noted.
3. Markdown files must follow the Goode Markdown Style Guide.
4. Tests should be created for each action or API endpoint.
5. Run Tests
6. Lint before committing
7. Do not include "CLAUDE" in the outputted code or documentation.
